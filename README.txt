# Customer Churn Prediction Project

This project was developed as part of the COMP 462 course and focuses on predicting customer churn using machine learning techniques.

## Contents

- `Main.py`: The main script for training and evaluating the churn prediction model.
- `Comp 462.pdf`: Project documentation including objectives, methodology, and findings.
- `ML rapor .pdf`: Detailed machine learning report with model analysis and results.

## Requirements

To run this project, ensure you have the following Python packages installed:

- pandas
- numpy
- scikit-learn
- matplotlib
- seaborn

You can install them using pip:

```bash
pip install pandas numpy scikit-learn matplotlib seaborn
